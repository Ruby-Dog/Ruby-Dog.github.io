C語言的指標
2021/06/06

什麼是指標(pointer)?

    當我們宣告一個變數時，程式會像記憶體要一塊空間，
而這個空間的起始位址就會被當作是變數的位址，也就叫做「指向該變數的指標(pointer)」。
    簡單來說，指標 (Pointer) 就是某變數的位址。

什麼是指標變數(pointer variable)?

    「指標(pointer)」不等於「指標變數(pointer variable)」，
指標變數(pointer variable)是用來存放指標的一種變數。
宣告的方式為： 
int*  pointer_TEST;
或: 
int  *pointer_TEST;
實例說明1: 
b = 2; 
//把2這個值給變數b
 
int* pointer; 
//跟記憶體要一塊區域稱為pointer,
//這塊區域專門放指向int型變數的指標(位址)
 
pointer = &b; 
//把變數b的地址值給pointer，注意不能寫成 pointer = b;
//因為 pointer 是專門存放地址的變數。
    
// 如此一來，我們就稱「指標變數 pointer 指向了變數 b」 !!
實例說明2: 
#include <stdio.h>

void main(){
    int c=60;
    int* pointer2 = &c;
    
    //可以用「*pointer」來拿到變數c。
    printf("c = %d \n",*pointer2);
    
    //!!重要!!
    /*這裡的「*」，和宣告指標變數的 int* pointer 的意義不太一樣。
    反而是和「&」相對應——「&」代表「取出地址」、
                         「*」代表「取出內容」。*/
                         
    /*所謂的「*pointer 取出的內容」指的到底是變數 b、還是變數 b 的值 2？
    這兩個是不同的東西喔！  
    變數 b 是這塊區域，2 是值。
      
    答案是：*pointer 代表的就是變數 b。所以我們可以把 *pointer 當作變數 b 來使用。*/        

    
}
結果:
c = 60
實例說明3: 
#include <stdio.h>

void main(){

    int d=3;
    int* pointer3 = &d;
        
    printf("變數d值 = %d \n",d);
    printf("變數d的位址 = %p \n",&d);
    printf("指標變數pointer3的值 = %p \n",pointer3);
        
    printf("\n");
        
    *pointer3=9999;
    printf("指標變數pointer3 = %d \n",*pointer3);
    printf("變數d的值 = %d \n",d);
    printf("指標變數pointer3位址 = %p \n",&pointer3);
    /*存放 pointer 這個變數的地址，
    和變數 b 的地址不一樣，
    所以利用「&pointer」後，
    可發現地址「0x7fff551b49c0」和變數 b 的地址果然不一樣。*/      

}
變數d值 = 3
變數d的位址 = 0x7ffdca914ef4
指標變數pointer3的值 = 0x7ffdca914ef4
指標變數pointer3 = 9999
變數d的值 = 9999
指標變數pointer3位址 = 0x7ffdca914ef8

「&a」與「*&a」

#include <stdio.h>

void main(){

    int a = 1;
    printf("變數a的值 a = %d\n",a);
    printf("變數a的位址 &a = %p\n",&a);   
}
結果:
變數a的值 a = 5
變數a的位址 &a = 0x7ffd4915661c
「&」為「位址運算子」 用來取得變數的位址。
在變數前加上「&」，便可取得該變數在記憶體中的位址。
「%p」是用來印出指標型態的值。 
#include <stdio.h>

void main(){

    int a = 1;

    printf("變數a的值 a = %d\n",a);
    printf("變數a的位址 &a = %p\n",&a);  
    printf("變數a的值 *&a = %d\n",*&a);

}
結果:
變數a的值 a = 5
變數a的位址 &a = 0x7ffd4915661c
變數a的值 *&a = 5
「 *」為「依址取值運算子」 用來取得指標所指向的變數內容。
在指標變數前加上*，表示以位址找到記憶體存放的值。
「*&b」和「b 」的意義是等價的。
